# 🔺 三角阵地战：新兵入伍指南

常言道：“兵马未动，粮草先行。”在这片由三角网格组成的战场上，**如果前方的部队失去了和总基地的联系，就会因为断水断粮而直接溃散消失！**

作为最高指挥官，你需要和敌军轮流派出士兵（占据新的网格点）。只要视线不被阻挡，你的新兵就会和原有的部队自动拉起一条“补给线”。

你的核心目标是**尽可能扩大你的安全领土**！
什么是领土？就是由你的士兵和补给线完完全全包围起来的区域。为了减轻前线的防御压力，聪明的参谋部会自动为你规划出**“最精简的防线”**——这条防线会尽可能缩短周长、减少多余的面积，但一定会把你所有的部队都保护在内，并且保证圈进来的地盘里**绝对没有**任何敌军或敌方补给线！

由于战备资源紧张，为了避免兵力浪费，**你不能把三个士兵紧挨着挤在一个小三角里**（禁止过度拥挤）。
但是，战场瞬息万变！**如果你这一步是直接踩在敌军的补给线上，切断了他们的粮草**，那就相当于抢夺了敌方的物资！这说明该地段战况极其激烈，此时你就可以打破常规，不受“不能拥挤”的限制，大举集结兵力发起猛攻！

最终，当双方都无兵可派、无地可占时，**谁圈出的领土面积更大，谁就是这场战争的最终胜利者！**
